Foldex はローカルストレージとリモート (SMB/SFTP/FTP/WebDAV) を横断的に扱えるファイラーです。広告なし・解析 SDK なし・運営者送信なしで、すべてが端末内で完結します。

★ ファイル管理
・ローカル: 内部ストレージ / SD / USB-OTG / Android/data (SAF 経由) / Termux ホーム (SAF tree)
・リモート: SMB / SFTP / FTP(S) / WebDAV
・HOME 画面でフォルダ・接続・機能をタイル化、長押しでドラッグ並べ替え
・検索 / ソート / 隠しファイル切替はフォルダごとに記憶
・ZIP の圧縮 / 解凍 (AES-256 パスワード対応)
・他アプリからの共有受信 (複数ファイル)

★ 内蔵ビューア
・画像 (ピンチズーム + スワイプ)
・PDF (ページキャッシュ + スクロールバー)
・テキストエディタ (Sora-editor、Canvas 描画、検索/折返し/行番号/Undo/Redo)
・Markdown / HTML (ソース ⇄ プレビュー切替)
・音声 / 動画 (リモートはストリーミング再生)

★ 同期 / サーバー
・SMB/SFTP/FTP/WebDAV との片方向 / 双方向同期
・スケジュール同期 (時刻指定 / 毎日 / 毎週 / 毎月)
・削除前の自動バックアップ (世代管理 + 容量しきい値)
・自機 SFTP / FTP サーバー (Argon2id 認証、ホスト鍵生成)
・実行ログ (アプリ内で閲覧)

★ プライバシー
・個人情報を一切収集・送信しません
・広告 SDK / 解析 SDK / Firebase は一切組み込まれていません
・ネットワーク通信はユーザーが設定した接続先との直接通信のみ
